# Ejercicio de Sistema de Productos Node.js

Práctica de tecnologías basadas en Node.js (Vue + Express). Sistema para la gestión de productos, categorías y usuarios.

> **Nota:** Este repositorio es un **ejercicio de práctica** y aprendizaje. El código está sujeto a cambios constantes, refactorizaciones y puede no estar completo.

## Sobre el proyecto

Este proyecto es una implementación académica para explorar la integración entre **Vue.js** (Frontend) y **Express** (Backend). El objetivo es construir un sistema modular que permita la administración de productos y usuarios, comprendiendo el flujo de datos en una arquitectura RESTful.

## Funcionalidades
* **Gestión de Productos:** Crear, listar, actualizar y eliminar productos.
* **Categorías:** Organización de productos por familias.
* **Usuarios:** Gestión básica de perfiles.
* **API:** Backend estructurado con Express y Node.js, implementando autenticación con JWT.
